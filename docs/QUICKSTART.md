# 快速上手 · QUICKSTART

> 给第一次打开「探索者 · Seeker」的你 —— 全程约 5 分钟。
>
> **只想先看看?** 打开 **[Web 演示版](https://aklmans.github.io/seeker/)** —— 无需安装,数据只存你的浏览器;真实 AI / 记忆 / 连接器在桌面端。

## 1 · 安装

### macOS

从 [Releases](https://github.com/aklmans/seeker/releases) 下载 `.dmg`,拖入「应用程序」。

**首次打开若提示「无法打开,因为无法验证开发者」**(App 暂未做 Apple 公证):
- 方式一:在「应用程序」里**右键(或按住 Control 点击)Seeker → 打开 → 打开**,只需一次;
- 方式二(终端):`xattr -cr /Applications/Seeker.app` 后正常双击。

### Windows

从 [Releases](https://github.com/aklmans/seeker/releases) 下载 `Seeker_*_x64-setup.exe` 安装。

**若弹蓝色 SmartScreen「Windows 已保护你的电脑」**(安装包暂未代码签名):点 **「更多信息」→「仍要运行」**,只需一次。Win10 首次启动若提示缺 WebView2,按提示装完即可(Win11 自带)。

> 以上是两家系统对未签名/未公证应用的统一提示,不代表检测到了问题。源码完全公开,可自行审计或从源码构建。

## 2 · 配好你的 AI(三条路,任选一条)

探索者是 **BYO(自带钥匙)** 设计:你填自己的模型端点与 Key,数据只发给你自己选择的服务商,项目方拿不到你的任何数据 —— 这也是「本地优先」的一部分。

打开 **数据设置 → 模型配置**:

| 路线 | 成本 | 步骤 |
|---|---|---|
| **A · 本地 Ollama(完全免费)** | ¥0,数据不出电脑 | 安装 [ollama.com](https://ollama.com) → 终端 `ollama pull qwen3` → 模型配置选 Ollama,填 `http://localhost:11434` |
| **B · 国内低价 API** | 每月几块钱量级 | DeepSeek / Kimi(Moonshot)/ 智谱等开放平台注册 → 创建 API Key → 选「OpenAI 兼容」,填对应 base URL + Key + 模型名 |
| **C · Anthropic / Gemini** | 按用量 | 各自控制台创建 Key → 选对应协议填入 |

> Key 只存进 macOS 钥匙串,不写入任何文件或日志;界面上你只会看到「已配置」。

## 3 · 第一句话说什么

界面中央就是 Agent。可以试试:

- 点快捷 chip「**智能匹配**」—— 看它怎么用你的数据干活(内置演示数据可直接玩);
- 输入 `/` —— 唤起命令面板,看看都有什么;
- 「帮我加一个岗位」—— 体验对话式录入;
- 到 **能力中心 → Skills** 建一个你自己的常用指令 —— 它会出现在输入框上方,一点即跑。

## 4 · 三个值得知道的设计

1. **应用可以关**:应用管理(左上角九宫格)里关掉「求职工作台」,它的页面、AI 权限、快捷项全部即刻下架 —— **数据保留**,随时可开回。
2. **隐私分层**:个人信息(姓名/电话/邮箱)存在独立区,AI 在结构上就读不到,不是「承诺不读」。
3. **破坏性操作**:AI 只能提议、你点确认才执行;删除均可撤销。

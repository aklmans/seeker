# 反馈模板 · FEEDBACK

> 感谢你试用「探索者 · Seeker」。不用写作文 —— 每题一两句、跳过不适用的即可。发 GitHub Issue 或直接微信我都行。

**1 · 你用它做了什么?**(哪怕只是点开看了看,也算)

**2 · 哪一步卡住了 / 让你困惑?**(具体到「点了什么、期待什么、实际看到什么」最有用)

**3 · 有没有哪个瞬间觉得「哎,这个有点意思」?** 是哪个功能?

**4 · 配 AI Key 那一步顺利吗?** 走的哪条路(Ollama / 国内 API / 其他)?卡了多久?

**5 · 一周后你还会打开它吗?** 会 / 不会 / 看情况 —— 为什么?

**6 · 如果只能留一个功能,你留哪个?如果必须砍一个,砍哪个?**

**7 · 其他任何想说的**(界面 / 文案 / 速度 / 稳定性 / 想要的功能……)

---

*环境信息(方便排查,可选):macOS 版本 / 芯片(Intel 或 Apple Silicon)/ 用的哪家模型*
